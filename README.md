# Smart Library Assistant

## Technologies Used

- Python
- OpenCV
- OCR (Tesseract)
- NLP
- Sentence Transformers
- FAISS
- RAG
- Machine Learning
- Decision Tree

## Features

1. Read book cover using OCR.
2. Ask questions from library documents.
3. Predict high-demand books.
4. Build vector database automatically.

## Install

pip install -r requirements.txt

## Run

python main.py
